Module 1 · Web-Grounded

Web Research & Brainstorming

~20 min

Copilot Chat can search the web to surface trends, research competitors, brainstorm ideas, and coach you through your day. Try these exercises to see how conversational follow-ups build on each response.

Exercise 1.1 Summarize Retail Industry Trends ▼
Try This Prompt
What have been the emerging trends in the retail industry over the past 4 months? Summarize using recent web articles and analyst reports.
Exercise 1.2 Market Research — New Product Ideas ▼

Start with research, then iteratively structure and enrich the output using follow-up prompts.

Starting Prompt
Identify trends in consumer preferences for grocery markets and use these insights to develop ideas for new products for our grocery market chain.
Follow-up
Structure those ideas in a table format and for each idea add a column for the potential target groups and ideas for an advertising campaign.
Follow-up
Add a column with the pros and cons of each idea.
Follow-up
How can I prioritize these ideas effectively?
💡 Tip
Each follow-up builds on the previous response — Copilot remembers context throughout the conversation, so you don’t need to repeat yourself.
Exercise 1.3 Competitor Research ▼
  1. 1 Replace [Tesla] with any company you’d like to research.
Try This Prompt
I work in marketing and focus on competitor research. Give me a concise summary of recent news about [Tesla]. Focus on web articles from the last 3 months. Provide the answer into two to three paragraphs and use a business tone.
Follow-up
Can you provide more details on sustainability?
Exercise 1.4 Daily Productivity Coach ▼
  1. 1 Replace [marketing manager] with your actual role.
Try This Prompt
Pretend you're my productivity coach for the day. What are three things I should prioritize based on my role as a [marketing manager]?
Exercise 1.5 Prepare for a Leadership Offsite ▼
Try This Prompt
Please help me prepare for a leadership offsite focused on team alignment and vision setting. I want to bring ideas that foster collaboration and long-term thinking. Use insights from McKinsey or the Center for Creative Leadership. Summarize 3–5 actionable suggestions I can share with the group.
Exercise 1.6 Creative Campaign Brainstorming ▼
  1. 1 Start with the brainstorming prompt, then provide context about your initiative using the second prompt.
Starting Prompt
Generate 10 creative campaign ideas for promoting our new sustainability initiative. The ideas should be suitable for social media and internal newsletters. Include a brief description for each.
Add Context
As a retailer, we're trying to reduce our carbon footprint from all points between design to manufacturing to warehousing to delivery to attract more eco-conscious customers to our brand.
Exercise 1.7 Copilot as a Thought Partner ▼
Starting Prompt
Brainstorm ideas for improving employee engagement in remote teams. Focus on low-cost, high-impact initiatives. Provide a list with short descriptions.
Add Your Team Context
Our team consists of around 50 people who work from home in a variety of cities across the US that extend coast-to-coast.
💡 Tip
Customise the team context with your own details — team size, locations, and working style — for more relevant suggestions.
Exercise 1.8 Respond to a Customer with Empathy ▼
  1. 1 Enter the instruction prompt first, then paste the customer inquiry.
Instruction
Write a response to this customer inquiry that is empathetic, clear, and resolves the issue. Include a greeting and closing.
Example Customer Inquiry
I really liked the jacket when I first received it. I thought it would be my best buy ever! Then after a few times wearing it, I noticed that the left-hand pocket seems shallow and not as deep as the right-hand pocket which seems like a craftsmanship issue. I also felt a tear in the shallow pocket and now it has expanded to where my finger-tip fits through it and I'm sure it's going to only get worse to the point that I'll put something in the pocket and will lose it through the hole!
Exercise 1.9 Transform Meeting Notes ▼
  1. 1 Attach your meeting notes file to the chat, then use the prompt below.
Try This Prompt
Summarize the attached meeting notes into a concise executive summary. Focus on key decisions, action items, and any unresolved issues. Format the output with bullet points and include a section for follow-up questions.
Follow-up
Can you help me with the first draft email with a brief executive summary for leadership?

Module 2 · File-Grounded

File-Based Analysis

~15 min

Copilot Chat can read and analyse files you attach to the conversation. Try these exercises using your own documents or the examples provided.

Exercise 2.1 Summarize Key Points from a Document ▼
  1. 1 Attach a document to the chat, then ask Copilot to summarize it.
Try This Prompt
Summarize the key points in the attached document.
Exercise 2.2 Cross-Reference Multiple Spreadsheets ▼

Copilot can compare data across multiple attached files to find discrepancies and generate recommendations. Attach your spreadsheets and try the prompt below.

Try This Prompt
Using the attached files, compare the invoice data with the contract data and identify any discrepancies in payment amounts or terms. Highlight any mismatched fields such as amounts, charges, or due dates. Generate a recommendation column indicating whether each item is complete and appropriate based on the contract terms. Create two separate sheets: one listing "Yes" recommendations and another listing "No" recommendations, including rationale for each decision.
💡 Tip
This kind of multi-file cross-referencing can save hours of manual comparison. Attach two or three related spreadsheets and let Copilot find the discrepancies.

Module 3 · Visuals

Data Visualisation

~10 min

Ask Copilot to create tables, charts, and other visuals from web data or your attached files — no spreadsheet skills required.

Exercise 3.1 Tables & Charts from Web Data ▼
Starting Prompt
Create a table of the top five countries by GDP from 2019 to 2024. List each country along with the GDP value for each given year.
Follow-up
Plot each country as a line in a line chart.
💡 Tip
You can go from raw data to a visual in just two prompts — first ask for structured data, then ask for a chart.
Exercise 3.2 Visualise Data from an Attached File ▼
  1. 1 Attach a spreadsheet with sales or numerical data, then use the prompts below.
Starting Prompt
Can you create a visual to represent the data in the attached file?
Follow-up Example
Create a pie chart that shows the distribution across categories.
📷
Module 4 · Vision

Image Analysis

~5 min

Copilot Chat can analyse images you attach and describe what it sees. Try it with photos, diagrams, or screenshots.

Exercise 4.1 Describe an Image ▼
  1. 1 Attach any image to the chat — a photo, diagram, or screenshot.
Try This Prompt
Can you please describe this image?
💡 Tip
Try follow-up questions like “What products are visible?” or “What improvements would you suggest for this store layout?” to explore the image further.
