Step 2

What Are AI Agents?

Concept AI Agents Defined ▼

AI Agents are autonomous systems that perceive their environment, make decisions, and take action to achieve goals. They go beyond simple chatbots by being able to reason, plan, use tools, and even collaborate with other agents.

Core capabilities:

  • Reasoning & Planning — Can break down complex tasks into steps
  • Tool Use — Call APIs, run workflows, search knowledge bases
  • Multi-Agent Collaboration — Agents can work together on complex problems
  • Memory — Remember context within a conversation and access enterprise data
Concept Three Types of AI Agents ▼
Type How It Works Best For
Declarative Agents Structured, rule-based responses FAQs, guided processes
Custom Engine Agents Business-logic-driven workflows Approvals, data processing
Autonomous Agents Self-directed, goal-oriented action Complex multi-step tasks

Step 4

How Agents Work — Under the Hood

Concept The Agent Processing Loop ▼
  1. 1 Trigger — A user message, event, or scheduled task kicks things off
  2. 2 Reasoning — The LLM interprets the request, selects tools, and plans steps
  3. 3 Tools & Actions — The agent calls APIs, searches knowledge bases, or runs Power Automate flows
  4. 4 Memory — Short-term context (conversation) + long-term storage (SharePoint, Dataverse)
  5. 5 Output — The agent responds, takes action, or hands off to a human
  6. 6 Human-in-the-Loop — Approval gates and escalation paths keep humans in control

Hands-On Build Your First Agent ▼
  1. 1 Log in to Copilot Studio at make.powerapps.com
  2. 2 Create a new agent and define its purpose (e.g., “HR onboarding assistant”)
  3. 3 Add a knowledge source — a SharePoint site or upload a document
