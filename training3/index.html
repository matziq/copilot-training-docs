Module 1 · Preparation

Demo Setup Instructions

~5 min

Prepare your environment before running the demos. Follow these guidelines to ensure a smooth, consistent experience for attendees.

Setup Environment & Account Requirements ▼
  1. 1 Use a separate Edge browser profile, logged in with a user from a CDX tenant (or another test tenant) without an M365 Copilot license assigned.
  2. 2 Using an account with a license may confuse attendees due to unavailable UI elements being visible such as Work/Web toggle, Notebooks, and agents like Researcher and Analyst.
  3. 3 Do not use your production Microsoft account — new features available internally first may visually differ or may not be available in customer tenants.
  4. 4 The demo user should not be associated to an agent billing plan (consumptive billing model). An account with consumptive billing access will show additional grounding options for Agent Builder and may cause confusion.
  5. 5 Place the following sample files in the test account’s OneDrive:
📝 Required Demo Files
• Drone Upselling Opportunities.DOCX
• Retail Scene.PNG
• Coffee Maker Meeting Transcription.DOCX
• EV Charger Demo Spreadsheet.XLSX
⚠️ Important
Ensure that Copilot Personalisation — Custom Instructions are cleared before starting your demo. You don’t want Copilot to respond unprofessionally, at least until you demo configuring it to use specific tone/voice/character.
🖥
Module 2 · UI Walkthrough

Copilot Chat Interface Overview

~10 min

Walk attendees through the complete Copilot Chat UI before diving into prompts.

Walkthrough UI Elements Tour ▼
  1. 1 Main UI: Point out the EDP Shield (when signed in), Central Prompt Pane, Work/Web Toggle (only if licensed — attendees with it should switch to Web for the demo), Prompt Dialog Box, and Starter Prompts.
  2. 2 Prompt Gallery: Click “+ More” to open the Copilot Prompt Gallery.
  3. 3 Left-hand Menu (5-in-1): Walk through Search, Chat, Agents, Notebooks (point out only if licensed), and Create (Conversations, Pages, Apps).
  4. 4 GPT Models: Show the various GPT versions available.
  5. 5 Settings: Upper-right ellipses → Personalisation → Custom Instructions and Copilot Memory.
  6. 6 Prompt & Response Options: Execute a prompt, then review Prompt Options (edit, copy, save, etc.) and Response Options (copy, thumbs, read aloud, etc.).
🔎
Module 3 · Web-Grounded

Prompting Samples

~25 min

Demonstrate how Copilot Chat uses web search to surface trends, perform competitor research, brainstorm ideas, and act as a thought partner. Each demo is a conversation that builds with follow-up prompts.

Demo 3.1 Summarize Retail Industry Trends ▼
Try This Prompt
What have been the emerging trends in the retail industry over the past 4 months? Summarize using recent web articles and analyst reports.
Demo 3.2 Perform Competitor Research ▼
Try This Prompt
I work in marketing and focus on competitor research. Give me a concise summary of recent news about [company name]. Focus on web articles from the last 3 months. Provide the answer in two to three paragraphs and use a business tone.
📝 Presenter Note
Example non-retail companies to substitute: Microsoft, Tesla, Apple, Citigroup, Boeing, Ford, Pfizer.
Follow-up
Can you provide more details on sustainability?
Demo 3.3 Act as Daily Productivity Coach ▼
Try This Prompt
Pretend you're my productivity coach for the day. What are three things I should prioritize based on my role as a [insert job title]?
📝 Presenter Note
Example retail roles to substitute: Store Operations Manager, Marketing Manager, Supply Chain Analyst, Talent Acquisition Specialist.
Demo 3.4 Help Brainstorming ▼
Starting Prompt
Generate 10 creative campaign ideas for promoting our new sustainability initiative. The ideas should be suitable for social media and internal newsletters. Include a brief description for each.
Context to Provide
As a retailer, we're trying to reduce our carbon footprint from all points between design to manufacturing to warehousing to delivery to attract more eco-conscious customers to our brand.
Demo 3.5 Copilot as a Thought Partner ▼
Starting Prompt
Brainstorm ideas for improving employee engagement in remote teams. Focus on low-cost, high-impact initiatives. Provide a list with short descriptions.
Add Team Context
Our team consists of around 50 people who work from home in a variety of cities across the US that extend coast-to-coast.
Demo 3.6 Respond with a Perspective ▼
Starting Prompt
Write a response to this customer inquiry that is empathetic, clear, and resolves the issue. Include a greeting and closing.
Customer Inquiry to Paste
I really liked the jacket when I first received it. I thought it would be my best buy ever! Then after a few times wearing it, I noticed that the left-hand pocket seems shallow and not as deep as the right-hand pocket which seems like a craftsmanship issue. I also felt a tear in the shallow pocket and now it has expanded to where my finger-tip fits through it and I'm sure it's going to only get worse to the point that I'll put something in the pocket and will lose it through the hole!
Demo 3.7 Transform Meeting Notes ▼
📝 Presenter Note
Use Coffee Maker Meeting Transcription.DOCX or paste meeting notes directly into the chat.
Try This Prompt
Summarize the following meeting notes into a concise executive summary. Focus on key decisions, action items, and any unresolved issues. Format the output with bullet points and include a section for follow-up questions.
Follow-up
Can you help me with the first draft email with a brief executive summary for leadership?

Module 5 · Visuals

Data Visualisation

~10 min

Show how Copilot Chat can create tables, charts, and visual representations of data from both web-sourced data and attached files.

Demo 5.1 Create Tables & Charts from Web Data ▼
Starting Prompt
Create a table of the top five countries by GDP from 2019 to 2024. List each country along with the GDP value for each given year.
Follow-up
Plot each country as a line in a line chart.
💡 Presenter Tip
This two-step sequence shows the progression from structured data (table) to visual representation (chart) — all within the same conversation.
Demo 5.2 Visualise Sales Data from a File ▼
📝 Presenter Note
Attach EV Charger Demo Spreadsheet.XLSX to the chat before running this prompt.
Starting Prompt
Can you create a visual to represent the sales data within this file?
Follow-up
Create a pie chart for the point in time of January 2024 that shows the distribution of sales across categories.
