Module 1 — 8:30–9:30 AM

What Are AI Agents?

🎯 Learning Outcomes
Understand agent fundamentals, types, and the Microsoft AI agent landscape.
Concept AI Agents Defined ▼

AI Agents are autonomous systems that perceive their environment, make decisions, and take action to achieve goals. They are built on large language models (LLMs) with enterprise data access.

Core capabilities:

  • Reasoning and planning
  • Tool use (APIs, connectors, workflows)
  • Multi-agent collaboration
  • Memory (short-term conversation + long-term knowledge)
Concept Three Agent Types ▼
Type Description
Declarative Agents Structured, rule-based responses
Custom Engine Agents Business-logic-driven workflows
Autonomous Agents Self-directed, goal-oriented action
Concept The Microsoft AI Agent Landscape ▼
Platform Description Audience
Microsoft 365 Copilot Everyday AI assistant across Word, Excel, Teams, Outlook All users
Copilot Studio Low-code/no-code builder with 1,000+ data-source connectors Business & IT teams
Azure AI Foundry Full-stack development with multi-agent orchestration Technical teams
Concept How Agents Work — Under the Hood ▼
  1. 1 Trigger — User message, event, or scheduled task initiates the agent
  2. 2 Reasoning — The LLM interprets the request, selects tools, and plans steps
  3. 3 Tools & Actions — Agent calls APIs, searches knowledge bases, runs Power Automate flows
  4. 4 Memory — Short-term context (conversation) + long-term (SharePoint, Dataverse)
  5. 5 Output — Agent responds, takes action, or hands off to a human
  6. 6 Human-in-the-Loop — Approval gates and escalation paths keep humans in control

🎯 Learning Outcomes

Lab Hands-On: Build Your First Agent ▼
📝 Trainer Tip
Walk through each step live on the projector. Pause after each step to let participants catch up. Have a co-facilitator roam the room to assist.
  1. 1 Log in to Copilot Studio at make.powerapps.com
  2. 2 Create a new agent and define its purpose
  3. 3 Add a knowledge source (SharePoint site or uploaded document)
  4. 4 Add one action using a Power Automate connector
  5. 5 Test in the built-in chat canvas
  6. 6 Refine and iterate based on test results

Module 3 — 10:30 AM–1:30 PM

Hackathon Challenge

🎯 Learning Outcomes
Apply agent-building skills to a real business problem using Microsoft Copilot Studio.
Briefing Challenge Brief & Ground Rules ▼
The Challenge
Build an AI Agent that solves a real business problem at your organization.

Ground Rules:

  • Teams of 3–5 people
  • Use Microsoft Copilot Studio (environments provided)
  • Agent must connect to at least one real data source or action
  • Demo must be live and working — no slides-only submissions
📝 Trainer Tip
Circulate between teams every 15–20 minutes. Help unblock technical issues early. Remind teams to focus on a working demo rather than a feature-complete product.
