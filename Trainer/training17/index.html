Overview

Session at a Glance

Audience & Objectives Who It’s For & What They’ll Learn ▼

This is a beginner-friendly, hands-on session introducing Microsoft 365 Copilot Chat (Basic) to people who may have never used an AI chat tool at work. Attendees do not need a Copilot license to use the product after the session.

Learning objectives

  • Explain what Copilot Chat is and how it differs from public AI tools.
  • Access Copilot Chat in the browser, in Microsoft 365 apps, and on mobile.
  • Write effective prompts using Goal, Context, Source, Expectations.
  • Use Copilot Chat across 18+ common scenarios: research, drafting, summarizing, visualizing, and creating.

Audience

  • Knowledge workers new to AI chat tools.
  • Mixed-role groups (sales, ops, marketing, finance, HR).
  • No Copilot license required to follow along.

Setup

Pre-Session Setup Checklist

Account & Browser Use a Clean Demo Account ▼
  • Use a separate Edge browser profile — not your daily profile.
  • Sign in with a test tenant account (e.g., a CDX tenant) that does NOT have a Copilot license assigned.
  • Do not use your production Microsoft account. Internal-first features and UI may differ from what customers see.
  • Avoid accounts linked to an agent billing plan — extra UI elements may appear and confuse attendees.
Demo Files Stage These in OneDrive ▼

Upload these sample files to the test account’s OneDrive so they’re available during demos:

  • Holiday Staffing Strategy.docx
  • Loyalty Program Meeting Transcript.docx
  • Quarterly Store Performance.xlsx
  • Retail Storefront.png
Reset Copilot Clean Starting State ▼
  • Open Settings (top-right ⋯) → Personalization.
  • Clear any Custom Instructions left over from a previous session.
  • Optionally clear Copilot Memory so the audience sees a clean starting state.
Tech Check Five Minutes Before You Start ▼
  • Confirm the green shield ✅ appears next to “New chat” — Enterprise Data Protection.
  • Run one prompt and one file upload end-to-end before the room arrives.
  • If you plan to demo image generation (Demo 16), make sure you haven’t already used today’s free image generation.
  • Project a high-contrast theme — the chat is busy with citations and toolbars.
  • Bump the browser zoom to 125–150% before screen sharing.

Deck

Deck Talking Points

Slides 1–18 Open, Frame, Tour ▼
  • Slides 1–3 — Open warm. Acknowledge that AI feels overwhelming — this session is the opposite. By the end they’ll have used Copilot Chat themselves.
  • Slide 4 — Learning outcomes. Set the expectation that this is hands-on.
  • Slides 6–7 — Value proposition. Anchor 30 minutes saved per day. Stress Enterprise Data Protection — this is the at-work version, not a public chatbot.
  • Slide 8 — Audience engagement. Optional: ask the room how they think it could help. Builds engagement before you show the product.
  • Slides 10–12 — Where to use it. Browser, M365 apps, mobile. Point out the green shield and what it means.
  • Slide 13 — Interface tour. Skip Notebooks — that’s license-only. Highlight the prompt box, suggested prompts, chat history, and the ⋯ menu.
  • Slides 14–15 — Models & Memory. Briefly explain GPT-5.X and Copilot Memory. Don’t get stuck in model details — the audience cares about outcomes, not models.
  • Slide 16 — Outlook. Mention Copilot in Outlook for licensed users. Skim past for unlicensed audiences (and see Demo 19 for the non-licensed Outlook story).
  • Slide 17 — App agents. Word/Excel/PowerPoint agents are licensed capabilities, but it’s good to know they exist.
  • Slide 18 — Demo gate. Switch to the browser.
Slides 19–36 Prompting, Features, Demos, Close ▼
  • Slides 20–24 — The art of prompting. Spend real time here. Run the vague-vs-detailed comparison live if you can.
  • Slides 25–29 — Features to know. File upload, data visualization, image generation, Prompt Gallery, Copilot Pages.
  • Slides 31–32 — Demo block. Pick demos that match the audience’s roles.
  • Slide 34 — Wrap-up. Prompt Gallery and community resources. Encourage daily use.
Framework The Four Prompt Ingredients ▼
💬 Trainer Note
Drill this on the prompting slides and then point it out in Demo 5. Every great prompt has these four ingredients.
  • Goal — what do you want Copilot to do?
  • Context — why do you need it, and who is involved?
  • Source — which information should Copilot use?
  • Expectations — what should the answer look or sound like?

Demos 1–4

Warm-Up & Research

Demo 1 · Warm-up Ask Copilot a Simple Question ▼

When to use: Anytime you need a quick explanation, definition, or summary. Files needed: None.

💬 Trainer Note
Use this as the gentle on-ramp. The point is to show Copilot Chat works like a conversation. Mention that plain language is fine.
🔎 Call Out
Point out the response toolbar that appears below Copilot’s answer — copy, regenerate, thumbs up/down, read aloud.
Prompt
Explain how a large language model works as if you’re explaining it to a curious 5th grader. Keep it under 150 words and use one short analogy.

Trainer tip: Try the same question without the word limit and the analogy to see how much your wording shapes the answer.

Demo 2 · Web research Research Industry Trends from the Web ▼

When to use: Staying current with what’s happening in your industry. Files needed: None.

💬 Trainer Note
Highlight that Copilot Chat (Basic) pulls from the web. Encourage attendees to try the same prompt with their own industry.
🔎 Call Out
Scroll the response and show the inline citation links — emphasize that you can click through to verify the sources.
Prompt
What have been the biggest trends in retail over the past 6 months? Summarize using recent web articles and industry analyst reports. Give me 5 trends with one short paragraph of context per trend.

Trainer tip: Switch “retail” to your industry — manufacturing, healthcare, financial services — and run it again.

Demo 3 · Iterate Competitor Research with a Follow-Up ▼

When to use: When you need to brief yourself before a meeting or pitch. Files needed: None.

💬 Trainer Note
Run the second prompt in the same chat to demonstrate context continuity. Try a different company if Patagonia returns sparse results — Microsoft, Tesla, Apple, Boeing, Ford, Pfizer, Costco are good options.
🔎 Call Out
Call out how “business tone” and the paragraph constraint shape the answer. Ask the room: “What changes if I delete those two phrases?”
Prompt
I work in marketing and focus on competitor research. Give me a concise summary of recent news about Patagonia. Focus on web articles from the last 3 months. Provide the answer in two to three paragraphs and use a professional business tone.
Follow-up (same chat)
Can you go deeper on anything they’ve shared about sustainability or supply chain practices?
Demo 4 · Role-play Make Copilot Your Productivity Coach ▼

When to use: Starting your day or rebooting your focus mid-week. Files needed: None.

💬 Trainer Note
Swap in your own job title live for color. The “pretend” framing is the lesson — assigning a role shapes tone and content.
🔎 Call Out
Note how the persona pattern (“Pretend you’re a…”) is reusable for a coach, editor, recruiter, financial planner, etc.
Prompt
Pretend you’re my productivity coach for the day. I’m a District Sales Manager covering 12 retail stores. What are three things I should prioritize today, and why? Keep each one short and practical.

Trainer tip: Swap in your real job title for a personalized version.

Demos 5–9

Writing & Communication

Demo 5 · Creativity Brainstorm Campaign Ideas ▼

When to use: Whenever you need 10 ideas in 60 seconds. Files needed: None.

💬 Trainer Note
Two-step flow showcases Copilot as a thinking partner: generate, then critique. The follow-up is where the real value lives.
🔎 Call Out
Highlight the four ingredients in the first prompt: goal (10 ideas), context (recycling program), source (customer segment), expectations (headline + description).
Prompt
Generate 10 creative campaign ideas for promoting our new in-store recycling program. The ideas should be suitable for social media and our customer email newsletter. Include a one-sentence description and a catchy headline for each. Our customers are eco-conscious shoppers ages 25–45 who already participate in our loyalty program.
Follow-up (same chat)
Pick the three you think are strongest and explain why. Then suggest one quick way to test each one.
Demo 6 · Writing Draft a Professional Email ▼

When to use: Sending the same kind of internal update you write all the time. Files needed: None.

💬 Trainer Note
The follow-up showcases tone-shifting in the same conversation. Ask the room: “When would you send the email version vs the chat version?”
🔎 Call Out
Show the response toolbar’s copy button and explain that paste works straight into Outlook or Teams.
Prompt
Draft an email to my team letting them know our weekly all-hands meeting is moving from Wednesday at 10am to Thursday at 2pm starting next week, because of a scheduling conflict with the regional review. Keep it warm, concise, and under 120 words. End with a friendly sign-off.
Follow-up (same chat)
Now rewrite it in a more casual tone, like a quick chat message.
Demo 7 · Refine Improve Your Writing ▼

When to use: When you’ve drafted something fast and want it sharpened. Files needed: None.

💬 Trainer Note
The “list what you changed” instruction is the highlight — it makes Copilot a writing tutor, not just a rewriter.
🔎 Call Out
Read the original and the rewrite out loud. Ask the audience which version they’d rather receive.
Prompt
Improve the writing of the paragraph below. Make it clearer, more concise, and more professional, but keep my voice friendly. Show me the rewritten version, then list 3 specific things you changed and why. "Hey team I wanted to reach out and let you all know that we are going to be making some changes to the way we are doing our weekly reports going forward I think the current process is kind of clunky and takes up too much time so I am hoping we can simplify it a bit, more details coming soon."
Demo 8 · Language Translate a Message ▼

When to use: Customer comms, multilingual teams, traveling teammates. Files needed: None.

💬 Trainer Note
Quick and crowd-pleasing. If you have native speakers in the room, ask them to sanity-check Copilot’s output.
🔎 Call Out
Mention this also works in reverse — paste a foreign-language email and ask Copilot to translate to English.
Prompt
Translate the message below into Spanish, French, and German. Keep the tone friendly and the meaning the same. Show each translation under its language name. "Thanks for stopping by our store this weekend! As a loyalty member, you've earned $10 off your next purchase. Use code THANKS10 at checkout — valid through the end of the month."
Demo 9 · Customer reply Respond to a Customer Complaint with Empathy ▼

When to use: Customer-facing teams, support, account managers. Files needed: None.

💬 Trainer Note
Show how “empathetic” shapes the tone vs a default reply. Optionally re-run with “firm and policy-driven” as a contrast.
⚠️ Reminder
Remind the room that the final reply should still be reviewed by a human — Copilot is a draft generator, not the brand voice.
Prompt
Write a response to the customer message below. The reply should be empathetic, clear, acknowledge the issue, and offer a path to resolution. Include a warm greeting and a professional closing. Keep it under 200 words. "I bought the running shoes online about six weeks ago. They felt great for the first week or two, but the stitching on the left shoe is already coming apart along the side. I bought these because of your brand's reputation and I'm pretty disappointed. What can you do to make this right?"

Demos 10–12

Summarize & Extract

Demo 10 · Summarize Summarize Pasted Meeting Notes ▼

When to use: Anytime you have rough notes from a meeting you want to send out. Files needed: None.

💬 Trainer Note
Two-step: first the summary, then the email. Both prompts run in the same chat.
🔎 Call Out
Point out the structure instruction (Decisions / Action Items / Open Questions) — Copilot follows formats reliably.
Prompt
Summarize the meeting notes below into a concise executive summary. Use three sections: Key Decisions, Action Items (with the owner in bold), and Open Questions. End with a "Follow-up Questions" section listing 3 things a leader might want to ask. NOTES: - Q4 budget review on Tuesday - Maria suggested reallocating $80K from print marketing to social ads — Devon agreed, will model the impact and come back next week - Loyalty program enrollment up 31% YoY — strongest growth segment is in-store sign-ups in the Southwest region - Tom raised that store associates are overwhelmed during peak weekend hours — Maria asked Ops to scope a 4-hour flex shift pilot - Holiday creative deck delayed by 1 week, new ETA Nov 18 — owned by the design team - Open: do we want to run a dormant-customer win-back offer before the new year? No owner assigned yet - Next meeting Nov 21
Follow-up (same chat)
Now turn that summary into a short email I can send to my leadership team.
Demo 11 · Attach file Summarize a Document You Upload ▼

When to use: Long memos, strategy docs, PDFs you don’t have time to read end to end. File needed: Holiday Staffing Strategy.docx.

💬 Trainer Note
Use the paperclip / Add content button in the prompt box. Drag the file directly from the desktop or pick it from the file picker. Confirm the green shield is visible.
🔎 Call Out
Supported uploads: Word, PowerPoint, Excel, PDF, and images. Note that very large files may be summarized only partially.
Prompt
Summarize this document in 5 bullet points a busy executive could read in 30 seconds. Then list the 3 most important risks and the 3 most important next steps, each as a single line.
Follow-up (same chat)
Draft a 4-sentence Teams message I can send to my regional VP recommending we approve this plan.
Demo 12 · Attach file Turn a Meeting Transcript into Action Items ▼

When to use: Post-meeting recap when you have a transcript but no time to write the email. File needed: Loyalty Program Meeting Transcript.docx.

💬 Trainer Note
Anyone who lives in meetings will love this one. Pair Demo 12 with Demo 10 to show that pasted notes and uploaded transcripts work the same way.
🔎 Call Out
Point out that the names of attendees come from the doc — Copilot didn’t invent them.
Prompt
From this meeting transcript, extract: (1) the key decisions made, (2) every action item with the owner’s name and any date mentioned, and (3) any open questions that don’t yet have an owner. Use a clean bulleted format.
Follow-up (same chat)
Now write a short recap email I can send to everyone who attended. Friendly tone, no more than 150 words.

Demos 13–14

Data & Visuals

Demo 13 · Attach file Visualize Data from a Spreadsheet ▼

When to use: Quick exploratory look at sales, performance, or any tabular data. File needed: Quarterly Store Performance.xlsx.

💬 Trainer Note
After Copilot renders the chart in the chat, hover over a bar to show interactivity. Mention you can copy the chart into PowerPoint or paste the data table into Excel.
🔎 Call Out
Charts are rendered as images inside the chat. They aren’t live links to the file.
Prompt
This spreadsheet shows quarterly sales by store. In plain English, what are the 3 most interesting patterns or insights? Then create a visual that compares annual totals by store, sorted from highest to lowest.
Follow-up (same chat)
Now show me the Q4 sales numbers grouped by region as a pie chart.
Demo 14 · Visualize Build a Table and Chart from Scratch ▼

When to use: You want a quick visual but you don’t have the data yet. Files needed: None.

💬 Trainer Note
No file needed — this demo also reinforces that Copilot Chat (Basic) can pull from the web.
⚠️ Reminder
Remind the room to verify the underlying numbers if they intend to use the chart in a real deliverable. Copilot pulls from public sources but isn’t a substitute for a sourced data feed.
Prompt
Create a table of the top 5 countries by GDP for each of the years 2020 through 2025. List each country along with the GDP value (in trillions of USD) for each year. Use the most recent reliable data you can find. Generate a real visual word cloud image that contains the most exported items for each country.
Follow-up (same chat)
Plot each country as a line in a line chart so I can see the trend over time.

Demos 15–16

Images

Demo 15 · Attach file Describe an Image ▼

When to use: Store visit reports, marketing reviews, accessibility descriptions. File needed: Retail Storefront.png.

⚠️ Trainer Note
Important: images must be uploaded or dragged in. They cannot be pulled in via the Context IQ “/” picker. Use the Add Content button or drag-and-drop.
🔎 Call Out
After the description, ask the room what they noticed in the image that Copilot did or didn’t mention.
Prompt
Describe what you see in this image. Then suggest two ways a store manager could use this view to improve the customer experience.
Demo 16 · Create visuals Generate an Image ▼

When to use: Quick training visuals, social posts, presentation cover slides. Files needed: None.

⚠️ Trainer Note
Save this for late in the demo block — Copilot Chat (Basic) is limited to one image generation per day for users without a Copilot license. If you’ve already generated an image earlier, demo it anyway and walk through what the result would look like.
🔎 Call Out
Talk about prompt specificity: “bright, photorealistic… warm and natural lighting… inclusive” are the levers that shape the result.
Prompt
Generate a bright, photorealistic image of a friendly retail store associate helping a customer in the apparel section of a modern clothing store. The lighting should be warm and natural, the background detailed and welcoming, and the scene should feel inclusive of a diverse range of customers. I’ll use this image in a training deck about excellent customer service.

Trainer tip: Got the result? Try asking Copilot to “make a second version with a different background” — but remember the daily limit.

Demos 17–18

Memory, Personalization & Fun

Demo 17 · Personalize Teach Copilot About You (Memory & Personalization) ▼

When to use: Once. After this, every future response is more relevant. Files needed: None.

💬 Trainer Note
Walk through the Memory settings live — top-right ⋯ menu → Personalization → Copilot Memory. Show the room what’s been saved.
🔎 Call Out
Memory is on by default. Tell users they can review and delete entries any time. Custom Instructions live in the same settings area.
Prompt 1 — Get to know me
Ask me 10 short questions about myself — my job, what I work on, what I find frustrating, how I like to communicate — to get to know me better so you can give me more relevant answers going forward. Ask them one or two at a time and wait for my reply. Remember the results for future interactions.
Prompt 2 — Learn my writing style
Ask me 5 questions to learn more about my writing style — things like tone, sentence length, words I use, and any phrases I want you to avoid. After I answer, summarize my style in a short paragraph I can save. Remember the results for future interactions.

Trainer tip: Want to review or clear what Copilot remembers? Click the ⋯ menu (top-right of Copilot Chat) → Personalization → Copilot Memory.

Demo 18 · Fun Lighten Up the Workday ▼

When to use: Kicking off a team meeting, brainstorming an offsite, or just for fun. Files needed: None.

💬 Trainer Note
Friendly closer. Optional but high-energy — leave time for one if Q&A is light. Reinforce that Copilot Chat isn’t only for serious work — the more you interact with it, the better you get at prompting.
Prompt
Suggest 5 office-friendly icebreaker questions I can use to kick off a team meeting next Monday morning. Mix one that’s a little quirky with four that are easy and inclusive.
Follow-up (same chat)
Now suggest 5 office-friendly jokes I could open the meeting with.

Troubleshooting

Common Snags When a Demo Misbehaves ▼
Symptom What to do
File upload fails or the file is too big. Trim the document or paste the relevant section as text into the prompt box. Demo 10 (pasted notes) is a great fallback.
Image generation says you’ve hit the daily limit. Copilot Chat (Basic) is limited to one image per day for users without a Copilot license. Pre-generate the image earlier in the day, or talk through the prompt and show the result you generated before the session.
Web response is sparse for your chosen company. Swap to a different example — Patagonia is reliable; Microsoft, Tesla, Apple, Costco, and Boeing are also strong. Avoid private companies with little news coverage.
Chart doesn’t render. Re-run the visualization prompt. If still failing, ask Copilot to “show the data as a table” instead — the data is still useful.
No green shield is visible. The user isn’t signed in to a work or school account. Sign out, switch profiles, and re-sign in with the correct account.
Audience asks about Notebooks, Researcher, or Analyst. Explain that those are part of the licensed Microsoft 365 Copilot experience. Today’s session focuses on what’s available to everyone in Copilot Chat (Basic).
Audience asks where their data goes. Reinforce Enterprise Data Protection: when signed in with the work account and the green shield is visible, conversation content is not used to train public models, and it’s protected by the same data commitments as other M365 services.

Wrap-Up Checklist

Close the Session Five Things to Hit in the Last Five Minutes ▼
  • Recap the four prompting ingredients (Goal, Context, Source, Expectations).
  • Point to the Prompt Gallery and the LinkedIn community.
  • Share the Student Companion Guide and demo files with attendees.
  • Encourage them to try Copilot Chat today — once before they go to sleep.
  • Collect feedback via the QR code on the closing slide.
Resources Bookmark These ▼

  • Settings → Personalization — manage saved memories, custom instructions, and chat history.
  • Prompt Gallery — under “See more” suggestions in Copilot Chat. Hundreds of ready-to-use prompts.
