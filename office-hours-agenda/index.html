Copilot Chat — Intro & Demos

Intro · ~4 min What Copilot Chat Is (and Why the Green Shield Matters) ▼
  • Where to find it: m365copilot.com, inside Teams/Outlook/M365 apps, or the M365 Copilot mobile app.
  • The green shield next to “New chat” confirms Enterprise Data Protection is on — your conversation stays inside your company’s protection boundary.
  • What’s free in Copilot Chat (Basic): file uploads, web grounding, charts & tables, one image per day, and Copilot Memory.
  • Prompting framework we’ll use: Goal · Context · Source · Expectations.

Block 2 · 16 min

Agents — Intro & Demos

Intro · ~3 min What an Agent Is (and What’s Free) ▼
  • Working definition: An agent is AI on a job description — instructions, personality, knowledge sources, and capabilities packaged together so the same task gets the same shape every time.
  • Where to find them: m365copilot.com → Agents in the left rail. The Agent Store lists Microsoft-built and partner agents.
  • What’s free for Copilot Chat (Basic) users: Microsoft-built agents (Prompt Coach, Idea Coach, Researcher preview, Writing Coach, Learning Coach), plus Agent Builder with web search, Image Generator, and Code Interpreter.
  • What needs a license: grounding on your SharePoint, OneDrive, email, calendar, or Teams data — that’s where the paid M365 Copilot license unlocks the next level.
🔎 Reference handout
The full Agent Revolution demo pack — including the “Hype Lab” build sheet — lives in the Agent Revolution — User Guide.

What we’ll look at: Two of the four Prompt Coach demos. Open Copilot Chat → Agents → search “Prompt Coach.”

Demo · Build Your Own Subtopic 3 — Build “The Hype Lab” in Agent Builder ▼

What we’ll look at: A speed-built declarative agent that any unlicensed Copilot Chat user can run. It’s a product/campaign concept generator for retail and CPG.

Block 3 · 5 min

Copilot Memory & Custom Instructions

Personalize Once, Benefit Daily Why This Is Worth 5 Minutes ▼

Copilot Memory lets Copilot Chat remember things about you across conversations — your role, the projects you work on, your writing voice, and preferences like “always answer in bullets” or “avoid hype words.” Custom Instructions are the standing brief you give Copilot up front so it never has to ask again. Together they turn every future response into a more personal, more relevant answer with no extra prompting cost.

What we’ll show live (one prompt, one settings tour):

  • Teach Run a quick “ask me 10 short questions about myself” prompt — Copilot interviews you, saves the memories, and uses them going forward.
  • Review Open the top-right ⋯ menu → Personalization → Copilot Memory to see, edit, or clear individual memories.
  • Set In the same Personalization area, drop in a few lines of Custom Instructions — who you are, how you like answers formatted, and what to avoid.
💬 The one-line version
Memory is what Copilot picks up about you over time. Custom Instructions is what you tell Copilot once, in writing, that it should always remember. Both live under Settings → Personalization.
