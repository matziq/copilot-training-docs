Module 1 · Preparation

Demo Setup Instructions

~5 min

Prepare your environment before running the demos. Follow these guidelines to ensure a smooth, consistent experience for attendees.

Setup Environment & Account Requirements ▼
  1. 1 Use separate Edge browser profiles, logged in with users from a CDX tenant (or another test tenant). Demos include examples using both accounts without and with an M365 Copilot license assigned.
  2. 2 Use an unlicensed account to illustrate the UX, the limits on Microsoft-provided agents, and the limited knowledge grounding options in Copilot Studio Lite.
  3. 3 Do not use your production Microsoft account — new features available internally first may visually differ or may not be available in customer tenants.
  4. 4 The unlicensed user should not be associated to a billing plan (consumptive billing model). An account with consumptive billing access will show additional grounding options and may cause confusion.
  5. 5 Due to session time constraints, pre-run the Researcher and Analyst demos and walk through them via the chat/conversation history (go to “All chats” and filter by agent name). You can rename your chats (right-click → Rename) to make finding them easier.
⚠️ Important
Start off using an account without an M365 Copilot license assigned!

Agent Mode Custom Agent in Copilot Chat ▼

Agent creation using only natural language in the “Describe” tab.

  1. 1 Enter the description prompt below in the Describe tab.
Agent Description
Create an agent to monitor public notices for consumer-based product recalls and summarizes "What do I need to know today?" for store ops and merchandising teams.
📝 Note
After just the initial guidance, a preview of the agent is already being displayed in the right-hand test pane.
  1. 2 Name the agent:
Name
Please name it "Total Recall"
📝 Note
The agent is auto-updated in the test pane after each change.
  1. 3 Refine the agent’s scope and behaviour:
Refinement
The agent should report on and surface recent consumer-based product recalls within the last 30 days. Please provide details on the recall reasons and recommended actions consumers should take. And please focus on recalls affecting the NW US area.
Source Guidance
Primarily use reputable websites like FDA, USDA, and CPSC but also information provided by affected retailers as well. Do NOT infer anything beyond published notices.
💡 Presenter Info
FDA = Food and Drug Administration · USDA = United States Department of Agriculture · CPSC = Consumer Product Safety Commission
  1. 4 Perform a couple of tests in the test pane, then add:
Additional Requirement
Add a requirement to cite all sources
  1. 5 Perform another test, then switch to the “Configure” tab and review the current configuration and options:

Module 3 · Premium

Premium Agent Demos

~20 min

These demos require an M365 Copilot license. Switch to your licensed account before proceeding. Researcher and Analyst should be pre-run and opened from chat history.

Premium Researcher ▼
⚠️ Pre-run Required
This example should be run ahead of time and opened from “Chats”.
Researcher Prompt
Build a white paper to help "early majority" companies create a plan to scale adoption of generative AI in the workplace via Microsoft 365 Copilot. This white paper should be written in a way that's easily digestible and practical for IT as well as business leaders. It should identify and address their fears and concerns, talk through ROI and ways to rationalize investments, tactical programmatic approaches, and should give very specific examples that can act as the art of the possible. Use technical press, recent academic research, analysts reports, and thought leadership written by industry leaders and management consulting firms.
Follow-up
Focus on large retail and consumer goods organizations.
